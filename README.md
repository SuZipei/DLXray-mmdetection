## Introduction

This project is based on the open source object detection toolbox [MMDetection](https://github.com/open-mmlab/mmdetection). The version is v3.3.0.

## Installation

Please refer to [Installation](https://mmdetection.readthedocs.io/en/latest/get_started.html) for installation instructions.
